По образованию я **кадастровый инженер (МИИГАиК, 2014)**. У меня **12-летний опыт работы профессиональным оценщиком**, а сейчас я применяю свои аналитические навыки в новой сфере — работаю **инженером-геодезистом на атомной станции**.

**Моя цель:** Научиться применять методы анализа данных и машинного обучения для автоматизации рутинных расчётов, обработки больших массивов измерительной информации и создания наглядных интерактивных отчётов в своей профессиональной деятельности.

## 🛠️ Технический стек & Навыки

**Язык программирования:** `Python`
**Основные библиотеки:** `Pandas`, `NumPy`, `Scikit-learn (sklearn)`, `Matplotlib`, `Seaborn`, `Hyperopt`, `Optuna`
**Инструменты и платформы:** `SQL`, `Git`, `Jupyter Notebook`, `VS Code`, `Comet.ml`
**Базы данных:** `PostgreSQL`

### 📊 Изученные ключевые направления
*   **Разведывательный анализ данных (EDA):** Введение в EDA, математическая статистика, проектирование признаков (Feature Engineering), применение статистических тестов, основы проектирования экспериментов (A/B-тестирование).
*   **Машинное обучение:** Теория МО, обучение с учителем (регрессия, классификация), обучение без учителя (кластеризация, понижение размерности), валидация данных и оценка моделей, отбор и селекция признаков (Feature Selection), оптимизация гиперпараметров, продвинутые методы МО.
*   **Математические основы:** Линейная алгебра в контексте линейных методов, математический анализ в контексте задач оптимизации.

**Текущий фокус изучения:** Углублённое изучение математики и статистики в ML (продолжение блока MATH&ML).
